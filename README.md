"Игры разума" - это набор из пяти консольных игр, созданных в рамках обучения на платформе Hexlet. Каждая игра задаёт пользователю вопросы, на которые нужно дать правильные ответы. Для победы необходимо правильно ответить на три вопроса подряд. Любой неверный ответ завершает игру.
## Архитектура проекта

Проект разработан с использованием принципов чистой архитектуры (выделение общего игрового движка) и принципа открытости/закрытости (ОСР). Вся общая логика (приветствие, опрос, валидация ответов) вынесена в единый движок, а каждая конкретная игра лишь поставляет данные для раундов.

## Минимальные требования

* **Node.js**: версия `v20.0.0` или выше
* **npm**: версия `10.0.0` или выше

## Установка и настройка

1. Клонируйте репорзиторий:
   ```bash
   git clone git@github.com:LoydSok/frontend-project-44.git
   cd frontend-project-44

   npm ci

## Список доступных игр и запуск

1. Знакомство (brain-games)
Простое приветствие пользователя, запрашивающее имя.

Запуск: brain-games

2. Проверка на чётность (brain-even)
Пользователю показывается случайное число. Нужно ответить yes, если число чётное, или no — если нечётное.

Запуск: brain-even

3. Калькулятор (brain-calc)
Пользователю показывается случайное математическое выражение (сложение, вычитание или умножение), которое нужно вычислить.

Запуск: brain-calc

4. Наибольший общий делитель (brain-gcd)
Пользователю показываются два случайных числа. Нужно вычислить и ввести их наибольший общий делитель (НОД).

Запуск: brain-gcd

5. Арифметическая прогрессия (brain-progression)
Показывается ряд чисел, образующий арифметическую прогрессию, в которой одно из чисел скрыто двумя точками (..). Нужно определить это число.

Запуск: brain-progression

6. Простое ли число? (brain-prime)
Пользователю показывается число. Нужно ответить yes, если число простое, или no — если оно составное.

Запуск: brain-prime
